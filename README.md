# Kode-Finale-Magang-GMRT-Week-2-3
Kode robot obstacle dengan arena pengujian terdiri dari dinding serta jalan tanpa cabang. Robot ditugaskan untuk dapat bergerak menggunakan dua roda aktif di belakang serta roda statis di bagian depan untuk mencapai tempat finish arena.

Beberapa parameter robot dalam kode belum ditune karena tidak sempat dilakukan pengujian penuh pada robot.
